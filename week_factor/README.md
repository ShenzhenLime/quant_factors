# A股周内效应因子策略构建（可复现）

> 构建人：**ShenzhenLime**  

---

## 1. 项目简介


基于Carhart 四因子模型（市场、市值、估值、动量），对个股日收益率做回归求得**日频残差**，再将**每周**第一个交易日的残差减去其余交易日的平均残差作为因子，以**周度**频率输出因子并进行回测评估。
$$R_{i,t} = \alpha_i + \beta_{i,MKT} MKT_t + \beta_{i,SMB} SMB_t + \beta_{i,HML} HML_t + \beta_{i,UMD} UMD_t + \epsilon_{i,t}$$
$$Factor_{i,week} = \epsilon_{i,first\_day} - \overline{\epsilon_{i,other\_days}}$$

项目使用 Python（pandas、numpy、joblib、matplotlib 等）实现，**代码与数据完整可运行**，并给出关键回测指标与图表。

## 2. 策略原理
在得到部分印证的半强有效市场假说下，股票价格反映了所有公开信息，但存在反应不足的情况。每天都有新的信息出现，但每一天的信息质量是不同的。

**理论逻辑**：周末相对于工作日的信息质量更高，因为周末的信息获取成本更高。具体到横截面上，每只股票的信息质量存在差异，通过周一的收益率与周二至周五的收益率差值能很好地刻画每只股票周末的信息质量。

**计算逻辑**：周一**特质收益率**高，代表这只股票的周一的收益率中存在较多无法被模型解释的部分。这可能来自于**质量高、市场反应积极**的信息。市场会在未来缓慢反应这个信息，由此为我们带来了超额收益。

1. **定价因子计算**（日频）：
   - 以 Tushare 的日线数据为基础，逐日计算 Carhart 四因子：
   - $MKT$：当日所有股票的平均收益率。
   - $SMB$：上一月底流动市值排名后三分之一组合收益 减 前三分之一组合收益。
   - $HML$：上一月底账面市值比前三分之一组合收益 减 后三分之一组合收益。
   - $UMD$：上一月底当月累积收益排名前三分之一组合收益 减 后三分之一组合收益。

2. **回归取残差**（单套 beta + 日频残差）：
   - 每只股票用全历史日收益率对四因子做一次 OLS 回归，得到一套 $\beta$（需≥220个交易日）。
   - 用该 $\beta$ 计算每个交易日的残差：
   $$\epsilon_{i,t} = R_{i,t} - (\hat\alpha_i + \hat\beta_{i,MKT} MKT_t + \hat\beta_{i,SMB} SMB_t + \hat\beta_{i,HML} HML_t + \hat\beta_{i,UMD} UMD_t)$$
   - 残差以日频存入 `stock_resids` 表。

3. **周内效应因子构建**（周频）：
   - 按自然周分组，取每周第一个实际交易日（兼容节假日无周一的情况）的残差 $\epsilon_{i,first}$，与其余交易日残差均值 $\overline{\epsilon_{i,other}}$ 之差作为因子：
   $$Factor_{i,week} = \epsilon_{i,first} - \overline{\epsilon_{i,other}}$$
   - 每周取最后一个交易日作为该因子的 `trade_date`，写入 `week_factor` 表。

4. **投资组合构建与回测**：
   - 周度调仓。
   - 将股票按因子值分为 `N=10` 组，做 **最高组多头、最低组空头** 的等权多空组合。
   - 计算 **IC/IR**、**分组收益**、**多空（L–S）组合收益**、与**基准收益**，并给出**夏普比率**、**累计收益曲线**等。
